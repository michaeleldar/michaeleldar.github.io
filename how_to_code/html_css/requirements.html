Linux

Linux has many very different version's
With many different text-editor's
But most have vi so that's what we will use.
To open terminal hold in the 'Ctrl' key and
the 'alt' key and while you are holding those
press 't'. Then type 'vi'.


Above is a picture of vi being used.
There are 3 modes in vi: command mode,
insert mode and last-line mode. When
you go into vi you are automaticly in
command mode. Command mode is the only
were you can switch into the other modes.
To go into command mode from any other mode
click 'Esc' (on your keboard). When you are in
command mode, click 'i' to go into insert mode
and ':' to go into last-line mode.
when you are in insert mode anything you type will
go onto the screen. When you are in last-line mode
there are 3 important commands:
'w' saves a File
'wq' saves and quits a file
'q!' quits a file without saving
